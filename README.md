# Childhood Depression Prediction in Malawi

## Overview
This project examines the determinants of childhood depression among children in Malawi using survey-based child, maternal, and household variables.

## Methods
- Data cleaning and preprocessing
- Feature engineering
- Logistic Regression
- CatBoost
- Threshold tuning
- SHAP and permutation importance
